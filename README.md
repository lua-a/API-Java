# API-Java
**CRUD para as entidades Order e Item**
#
**Objetivo**

- Essa API tem como objetivo desenvolver um Crud - Create (criar), Read (ler), Update (atualizar) e Delete (apagar) - para as tabelas de Item e Order.

##
**Tecnologias**

- Java
- Spring boot
- Spring Data Jpa
- Spring MVC


##
**Ambientação**

- IDE
- Insomnia
- PgAdmin
- JDK 11
